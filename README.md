## Technology Used

- kotlin coroutine & flow
- Caching using room
- Retrofit http client
- junit5 unit testing
- espresso ui testing

### You can manually build this app

```
#clone
$ git clone https://github.com/prasetyodidi/sepatuku.git

open on android studio and run app, or build your apk file
```
